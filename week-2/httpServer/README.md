# 🧠 Kidney Health API (Express.js)

This is a simple REST API built using Express.js that simulates kidney health management for a user.
It demonstrates basic CRUD operations (Create, Read, Update, Delete) using an in-memory data structure.

---

## 🚀 Features

- Get total, healthy, and unhealthy kidneys
- Add a new kidney (healthy/unhealthy)
- Mark all kidneys as healthy
- Delete unhealthy kidneys
